# Security Policy

## Supported Versions

This project is maintained from the `main` branch. Use the latest commit for fixes.

## Reporting A Vulnerability

Please report security issues privately to the repository owner before publishing details. Include:

- A concise description of the issue.
- Steps to reproduce.
- Affected platform and build configuration.
- Any relevant logs or crash details.

The project has no formal security response SLA, but reports should receive an acknowledgement before public disclosure.
